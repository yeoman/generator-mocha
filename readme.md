# Mocha generator [![Build Status](https://secure.travis-ci.org/yeoman/generator-mocha.svg?branch=master)](http://travis-ci.org/yeoman/generator-mocha)

## Install

```
$ npm install --global generator-mocha
```


## Usage

```
$ yo mocha
```
